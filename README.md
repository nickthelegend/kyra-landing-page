## Tech Stack

**Next.js** · **React** · **Tailwind CSS** · **TypeScript**

## Getting Started

```bash
# clone
git clone https://github.com/nickthelegend/kyra-landing-page.git
cd kyra-landing-page

pnpm install
pnpm run dev
pnpm run start
pnpm run build
```
